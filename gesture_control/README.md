# Presento - Intuitive Hand Gesture Controller

This project allows you to control presentation slides and video playback using intuitive hand gestures.

## Features

- **Start/Resume:** Open Palm
- **Exit:** Closed Fist
- **Volume Control:** Thumbs Up (Volume Up) / Thumbs Down (Volume Down)
- **Brightness Control:** Thumb Pointing Right (Increase Brightness) / Thumb Pointing Left (Decrease Brightness)
- **Navigation:** 
  - Moderate Swipe Right → Next Slide / Move 10 sec Forward (Video)
  - Moderate Swipe Left → Previous Slide / Move 10 sec Backward (Video)
- **Video Playback:** 
  - Fast Swipe Right → Fast Forward (Video)
  - Fast Swipe Left → Rewind (Video)
- **Scrolling:** Pointing Gesture (scroll vertically; add if desired)
